# World Happiness Report Analysis

## Project Overview
This project analyzes the World Happiness Report dataset using Python, Plotly, and Streamlit. It explores factors affecting happiness through interactive visualizations and an interactive dashboard.

## Files
- raghu.ipynb – Jupyter Notebook
- app.py – Streamlit Application
- World Happiness Report.csv – Dataset
- Raghu PPT.pptx – Presentation
- requirements.txt – Required Python packages
